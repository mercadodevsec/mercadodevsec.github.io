## 🎯 Overview

This is a personal portfolio website designed to present my professional profile as an IT Senior at the University of North Florida. The site features a fixed sidebar with profile information and a centered main content area that displays my education, projects, skills, and certifications.

---

## ✨ Features

- **Fixed Sidebar Navigation**: Left-aligned profile card with photo, name, title, and social links that stays in place while scrolling
- **Responsive Design**: Automatically adapts to mobile devices with a stacked layout
- **Clean Typography**: Modern sans-serif fonts with good readability
- **Custom Color Scheme**: Professional dark sidebar with light main content area
- **Social Media Integration**: Quick-access buttons for GitHub, LinkedIn, and Email
- **Project Showcase**: Detailed project descriptions with technology tags
- **Education & Certification Display**: Clearly organized academic and professional credentials
- **Skills Section**: Comprehensive list of technical competencies

## 🚀 Installation

### Option 1: GitHub Pages (Recommended)

1. Fork or clone this repository
2. Rename the repository to `username.github.io` (replace `username` with your GitHub username)
3. Replace `profile.jpg` with your own profile photo
4. Update the personal information in the HTML
5. Push to GitHub
6. Your site will be live at `https://username.github.io`
